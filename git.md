# iTerm编辑器
⌘ + Enter: 切换全屏
⌘ + D: 垂直分屏，⌘ + shift + d: 水平分屏
⌘ + T: 打开一个新标签页
⌘ + W: 关闭当前标签页
ctrl + U: 清空当前行
⌘ + R: 清屏，其实是滚到新的一屏，并没有清空。等价于ctrl + l

⌘ + shift + H: 剪贴板历史
